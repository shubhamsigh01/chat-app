# Orbit Chat - Real-time Messaging App

A premium, real-time chat application built with React, Node.js, Express, and Socket.io.

## Features
- Real-time messaging with Socket.io
- Multiple chat rooms
- Premium Glassmorphism UI
- Responsive design
- Smooth auto-scroll to bottom

## Project Structure
- `server/`: Node.js + Express + Socket.io backend
- `client/`: React + Vite + Socket.io-client frontend

## How to Run

### 1. Start the Server
```bash
cd server
node index.js
```
The server will run on `http://localhost:3001`.

### 2. Start the Client
```bash
cd client
npm run dev
```
The client will run on `http://localhost:5173`.

## Technologies Used
- **Frontend**: React, Vite, Socket.io-client, CSS (Vanilla)
- **Backend**: Node.js, Express, Socket.io, CORS
